# healthcare-dashboard-powerbi



📌 Project Overview

This project is a Healthcare Data Analytics Dashboard created using Power BI.
The dashboard helps analyze patient records, hospital performance, billing insights, test results, and admission trends through interactive visualizations.

The main goal of this project is to transform raw healthcare data into meaningful insights for better decision-making.



📊 Dashboard Features
Patient Admission Analysis
Billing Amount Overview
Test Result Analysis
Blood Group Distribution
Gender-wise Patient Analysis
Doctor & Department Insights
Length of Stay Analysis
Interactive Filters & Slicers


🛠 Tools & Technologies Used
Power BI
Power Query Editor
Data Cleaning & Transformation
Data Visualization




🧹 Data Cleaning Performed

The following data cleaning and preprocessing steps were done using Power Query Editor:

Removed null values
Changed data types
Renamed columns
Rounded billing amounts
Removed duplicates
Formatted dates
Created calculated columns for analysis
Cleaned inconsistent values



📈 Key Insights
Identified highest billing departments
Analyzed patient admission trends
Compared test results across blood groups
Observed gender distribution among patients
Evaluated hospital stay durations
